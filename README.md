<h1>tetris</h1>
 VB6 visual basic 6.0 俄罗斯方块 tetris
<h3>操作按键</h3>
<pre>
开始    鼠标左键
暂停    鼠标右键
左      键盘左/A
右      键盘右/D
软降    键盘下/S
硬降    C
逆旋转  空格键/Z
顺旋转  X
<b>如果有什么建议，可以给我留言^.^</b>
 </pre>
